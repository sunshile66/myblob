# 纯前端图片编辑器使用说明

## 🎨 简介

这是一个完全基于前端的图片编辑器，使用 Vue 3 + fabric.js 实现，无需任何后端支持。

**特点**：
- ✅ 纯前端实现，无需后端 API
- ✅ 所有操作本地完成，保护隐私
- ✅ 实时响应，无网络延迟
- ✅ 支持离线使用
- ✅ 完全免费

## 🚀 快速开始

### 访问方式
```
http://localhost:3001/tools/image-editor
```

### 从工具首页进入
```
http://localhost:3001/tools
```
然后点击"图片编辑"卡片

## 📖 功能说明

### 1. 基础操作

#### 打开图片
- 点击顶部"打开"按钮
- 选择本地图片文件
- 支持 JPG、PNG、GIF、WebP 等格式
- 图片会自动调整到合适大小

#### 保存图片
- 点击"保存"按钮
- 直接下载为 PNG 格式
- 或使用"导出"选择格式（PNG/JPEG/WebP）

#### 撤销/重做
- 点击左上角的撤销/重做按钮
- 或使用快捷键：
  - `Ctrl + Z` - 撤销
  - `Ctrl + Y` - 重做

### 2. 工具使用

#### 选择工具 (V)
- 选择和移动对象
- 调整对象大小
- 旋转对象

#### 裁剪工具 (C)
1. 点击裁剪工具或按 C 键
2. 拖动绿色边框选择裁剪区域
3. 调整边框大小和位置
4. 按 Enter 键或点击"确认裁剪"
5. 按 Escape 键或点击"取消"退出

#### 画笔工具 (B)
- 自由绘画
- 可调整颜色（前景色）
- 可调整画笔大小

#### 橡皮擦 (E)
- 擦除不需要的部分
- 可调整擦除大小

#### 文字工具 (T)
1. 点击文字工具
2. 在画布上点击
3. 输入文字
4. 可调整字体大小、颜色

#### 形状工具
- 矩形
- 圆形
- 三角形
- 点击形状后在画布上拖动绘制

### 3. 图像调整

#### 亮度/对比度/饱和度
- 在右侧"调整"面板
- 拖动滑块调整参数
- 实时预览效果

#### 滤镜效果
- 在右侧"滤镜"面板
- 点击预设滤镜直接应用
- 包含 8 种常用滤镜

### 4. 图层管理

#### 图层操作
- 查看右侧"图层"面板
- 点击图层选中
- 点击眼睛图标显示/隐藏
- 可调整图层顺序

#### 添加图层
- 添加文字会自动创建文字图层
- 添加形状会自动创建形状图层
- 可叠加多个元素

### 5. 视图控制

#### 缩放
- **鼠标滚轮**: 向上放大，向下缩小
- **快捷键**: 
  - `+` 或 `=` - 放大
  - `-` - 缩小
  - `0` - 适应屏幕
- **按钮**: 点击缩放按钮

#### 平移
- 按住空格键 + 拖动鼠标
- 或使用选择工具拖动画布

### 6. 快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl + S` | 保存 | 快速保存为 PNG |
| `Ctrl + Z` | 撤销 | 撤销上一步操作 |
| `Ctrl + Y` | 重做 | 重做被撤销的操作 |
| `Delete` | 删除 | 删除选中对象 |
| `V` | 选择工具 | 切换到选择工具 |
| `C` | 裁剪工具 | 切换到裁剪模式 |
| `B` | 画笔工具 | 切换到画笔 |
| `E` | 橡皮擦 | 切换到橡皮擦 |
| `T` | 文字工具 | 切换到文字工具 |
| `+` / `=` | 放大 | 放大视图 |
| `-` | 缩小 | 缩小视图 |
| `0` | 适应屏幕 | 自动调整显示大小 |
| `Enter` | 确认裁剪 | 应用裁剪区域 |
| `Escape` | 取消裁剪 | 退出裁剪模式 |

## 💡 使用技巧

### 技巧 1：快速裁剪
```
1. 打开图片
2. 按 C 键
3. 拖动选择区域
4. 按 Enter 确认
```

### 技巧 2：精细编辑
```
1. 鼠标滚轮放大细节
2. 使用画笔工具编辑
3. Ctrl+Z 撤销不满意的操作
```

### 技巧 3：多图层的妙用
```
1. 先添加背景
2. 添加文字（自动新图层）
3. 添加形状装饰
4. 分别调整每个图层
```

### 技巧 4：滤镜组合
```
1. 先调整亮度/对比度
2. 再应用滤镜
3. 可以叠加多个效果
```

### 技巧 5：高效保存
```
- 编辑过程中按 Ctrl+S 随时保存
- 完成后用导出功能选择格式
- 网络图片用 WebP 格式更小
```

## ⚠️ 注意事项

### 1. 浏览器兼容性
- 推荐使用 Chrome、Edge、Firefox
- 需要支持 Canvas API
- 建议使用最新版浏览器

### 2. 性能建议
- 图片大小建议不超过 5MB
- 图层数量不宜过多（建议<20）
- 大图片编辑时可能稍慢

### 3. 数据保存
- 所有操作在本地完成
- 关闭页面会丢失未保存的内容
- 建议随时按 Ctrl+S 保存
- 重要作品及时导出

### 4. 格式支持
- **导入**: JPG, PNG, GIF, WebP, BMP
- **导出**: PNG, JPEG, WebP
- 透明背景请用 PNG 格式

## ❓ 常见问题

### Q: 图片加载失败？
A: 检查图片格式是否支持，或图片文件是否损坏。

### Q: 操作卡顿？
A: 图片太大或图层太多，建议简化操作。

### Q: 如何恢复误删的内容？
A: 按 Ctrl+Z 撤销，或重新打开图片。

### Q: 导出的图片不清晰？
A: 导出时选择 PNG 格式，质量更好。

### Q: 可以批量处理吗？
A: 当前版本不支持，需要一张一张处理。

### Q: 手机能用吗？
A: 可以，但建议使用平板或电脑获得更好体验。
