## 📧 邮箱配置

### 1. QQ邮箱配置
1. 登录QQ邮箱 → 设置 → 账户
2. 开启POP3/SMTP服务
3. 获取授权码（不是QQ密码）
4. 在 `.env` 文件中配置：
```env
EMAIL_BACKEND=django.core.mail.backends.smtp.EmailBackend
EMAIL_HOST=smtp.qq.com
EMAIL_PORT=587
EMAIL_USE_TLS=True
EMAIL_HOST_USER=your-email@qq.com
EMAIL_HOST_PASSWORD=your-authorization-code
DEFAULT_FROM_EMAIL=your-email@qq.com
```

### 2. 163邮箱配置
```env
EMAIL_HOST=smtp.163.com
EMAIL_PORT=465
EMAIL_USE_SSL=True
```

### 3. Gmail配置
```env
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USE_TLS=True
```

## 📱 短信配置（阿里云短信）

### 配置步骤：
1. 登录阿里云控制台 → 短信服务
2. 创建签名和模板
3. 获取AccessKey
4. 在 `.env` 文件中配置：
```env
ALIYUN_ACCESS_KEY_ID=your-access-key-id
ALIYUN_ACCESS_KEY_SECRET=your-access-key-secret
ALIYUN_SMS_SIGN_NAME=MyBlob
ALIYUN_SMS_TEMPLATE_CODE=SMS_123456789
```

## 🔐 第三方登录配置

### 1. 微信登录
1. 访问 https://open.weixin.qq.com/
2. 创建应用 → 网站应用
3. 获取 AppID 和 AppSecret
4. 配置授权回调域：`localhost:3001`
5. 在 `.env` 文件中配置：
```env
WECHAT_APP_ID=your-wechat-app-id
WECHAT_APP_SECRET=your-wechat-app-secret
```

### 2. QQ登录
1. 访问 https://connect.qq.com/
2. 创建应用
3. 获取 AppID 和 AppKey
4. 配置回调地址
5. 在 `.env` 文件中配置：
```env
QQ_APP_ID=your-qq-app-id
QQ_APP_SECRET=your-qq-app-secret
```

### 3. 微博登录
1. 访问 https://open.weibo.com/
2. 创建网站应用
3. 获取 App Key 和 App Secret
4. 配置授权回调页
5. 在 `.env` 文件中配置：
```env
WEIBO_APP_ID=your-weibo-app-id
WEIBO_APP_SECRET=your-weibo-app-secret
```

### 4. GitHub登录
1. 访问 https://github.com/settings/developers
2. 创建 OAuth App
3. 获取 Client ID 和 Client Secret
4. 配置 Authorization callback URL
5. 在 `.env` 文件中配置：
```env
GITHUB_CLIENT_ID=your-github-client-id
GITHUB_CLIENT_SECRET=your-github-client-secret
```
