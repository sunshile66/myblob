## 🚀 快速启动指南

### 后端启动

```bash
cd myblob-backend
mvn spring-boot:run
```

访问 http://127.0.0.1:8000

### 前端启动

```bash
cd blog-frontend
pnpm dev
```

访问 http://localhost:3001 或 http://localhost:3002

## 📈 数据库表统计

总计 **42+** 张数据表：

- 基础实体: BaseEntity (所有表通用)

- 评论/互动: Comment, CommentLike, CommentReaction, Emoji, Sticker
