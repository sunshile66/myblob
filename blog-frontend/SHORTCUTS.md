# 图片编辑器快捷键说明 🎨

## 常用快捷键

### 文件操作
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl + S` | 保存图片 | 快速保存为 PNG 格式 |
| `Ctrl + O` | 打开图片 | 打开文件选择对话框 |

### 编辑操作
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl + Z` | 撤销 | 撤销上一步操作 |
| `Ctrl + Y` | 重做 | 重做被撤销的操作 |
| `Delete` / `Backspace` | 删除 | 删除选中的对象 |

### 视图缩放
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `+` / `=` | 放大 | 放大画布视图 |
| `-` | 缩小 | 缩小画布视图 |
| `0` | 适应屏幕 | 自动调整到最佳显示大小 |
| **鼠标滚轮** | 缩放 | 滚动鼠标滚轮快速缩放 |

### 裁剪功能
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `C` | 裁剪工具 | 切换到裁剪模式 |
| `Enter` | 确认裁剪 | 应用裁剪区域 |
| `Escape` | 取消裁剪 | 退出裁剪模式 |

### 工具切换
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `V` | 选择工具 | 选择和移动对象 |
| `B` | 画笔工具 | 自由绘画 |
| `E` | 橡皮擦 | 擦除内容 |

## 裁剪功能使用说明

### 步骤：
1. **选择裁剪工具**
   - 点击工具栏的裁剪按钮，或按 `C` 键

2. **调整裁剪区域**
   - 拖动绿色边框的控点调整大小
   - 拖动边框内部移动裁剪区域
   - 裁剪区域会显示为半透明

3. **确认或取消**
   - 按 `Enter` 键或点击"确认裁剪"按钮应用裁剪
   - 按 `Escape` 键或点击"取消"按钮退出裁剪

### 提示：
- 裁剪时会禁用其他对象的选择和移动
- 裁剪后会自动保存历史记录，可以撤销
- 裁剪后的画布会自动调整到合适大小

## 缩放功能说明

### 鼠标滚轮缩放
- 将鼠标移动到画布上
- 向上滚动放大，向下滚动缩小
- 缩放会以鼠标位置为中心
- 缩放范围：10% - 500%

### 适应屏幕
- 加载图片后会自动调整到最佳显示大小
- 按 `0` 键可以随时恢复到适应屏幕的大小
- 图片会自动限制在合理范围内（最大 1200x800，最小 600x400）

## 工具提示

### 左侧工具栏
- **选择工具 (V)**: 选择和移动对象
- **裁剪 (C)**: 裁剪图片
- **画笔 (B)**: 自由绘画
- **橡皮擦 (E)**: 擦除内容
- 以及更多工具...

### 顶部操作栏
- 显示当前文件名和尺寸
- 裁剪时显示确认/取消按钮
- 显示同步状态

### 右侧面板
- **图层**: 管理图层可见性和顺序
- **调整**: 亮度、对比度等参数
- **滤镜**: 预设滤镜效果

## 使用技巧

1. **快速保存**: 编辑完成后直接按 `Ctrl + S`
2. **快速撤销**: 操作失误立即按 `Ctrl + Z`
3. **细节编辑**: 使用鼠标滚轮放大细节
4. **整体预览**: 按 `0` 键查看全貌
5. **精确裁剪**: 使用裁剪工具 + Enter 快速确认

## 注意事项

⚠️ **重要提示**:
- 裁剪前请确保选择了正确的区域
- 删除操作不可恢复（除非使用撤销）
- 定期保存重要作品
- 大图片会自动缩小显示，但不影响导出质量
